# ReFleek — Run & Deploy

Node-based circular design-to-production workflow. Pattern artwork and product
renders are generated by Google Gemini (`google/gemini-2.5-flash-image`) via
OpenRouter; the tech pack exports to PDF with the artwork + render embedded.

## Architecture

```
Browser (UI/ReFleek.dc.html)
  → fetch('/api/generate')          ← same-origin, no CORS, key never exposed
    → api/generate.js (serverless)  ← reads OPENROUTER_API_KEY from env
      → OpenRouter → Gemini 2.5 Flash Image
```

Without `OPENROUTER_API_KEY` the endpoint returns deterministic **mock** SVG
artwork so the whole flow is demoable offline.

## Run locally

```bash
# mock mode (no key needed)
node dev-server.cjs
# → http://localhost:3939

# live generation
OPENROUTER_API_KEY=sk-or-xxxxx node dev-server.cjs
```

Open http://localhost:3939 — it redirects to `/UI/ReFleek.dc.html` (needed so
the page's `./support.js` reference resolves). Requires internet for the React
runtime (loaded from unpkg by `support.js`).

## Deploy to Vercel

1. Push this repo to GitHub (already on `origin`).
2. Import the repo in Vercel — no build step; `api/generate.js` is auto-detected
   as a serverless function, `vercel.json` redirects `/` → the UI.
3. In **Project → Settings → Environment Variables** add:
   - `OPENROUTER_API_KEY` = your OpenRouter key
   - *(optional)* `OPENROUTER_IMAGE_MODEL` to override the default model
4. Deploy. Visit the URL → the Pattern and Render nodes now call real Gemini.

## Craft constraint rules (surface finishes)

| Material stream            | Embroidery | Digital print | Laser etching |
|----------------------------|:----------:|:-------------:|:-------------:|
| Unfinished deadstock       |     ✅     |      ✅       |      ✅       |
| Upcycled reclaimed fabric  |     ✅     |   ✕ blocked   |      ✅       |

Selecting an upcycled material auto-removes digital print and shows the reason;
the print chip stays visible but locked in the Technique node.

## Demo path (verified end-to-end in mock mode)

Product → Material → Technique (print locks/unlocks by material) → Pattern
(Generate artwork) → Layout → Render (Generate AI render) → Tech Pack
(Export PDF, ~2 MB with images embedded) → Producer.
